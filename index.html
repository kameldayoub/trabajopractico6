  2. ¿Cómo se integra JavaScript a HTML?.
    • La integracion de JS al HTML se puede hacer de dos formas diferentes (pero en ambos se utiliza la etiqueta "script") :

    • -Por el body, esta foma se deve de declarar antes del cierre del body y despues del contenido hecho en html:

    • < script type=" text/javascript">
      var saludo = "hola";
      < /script>

    • -Por el head, se puede utiliza el atributo src para declarar la ubicacion del archivo js o dentro de la etiqueta se escribi lo que se quiere hacer con el script:

    • < script src="js/script.js">< /script>
      < script> console.log(); < /script>
  3. ¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?.
    • Manejo de variables:
      JS permite definir 3 tipos de variables let, const y var. La diferencia entre let y const es que el primer tipo se le puede asignar el valor y a const por otra parte no. Ejemplo:

      var numero1= 1;
      let numero2= 2;
      const numero3= 3;

      Tipos de datos:
      JS al ser un lenguaje dinámico este ya puede saber el tipo de datos según lo que contenga la variable en su interior. Si por ejemplo yo coloco la siguiente linea "var numero1= 1;" (anteriormente nombrada), sabe de antemano al contener un número, tomará el tipo de dato como numérico, lo mismo si contiene comillas "const nombre= "kamel";", entenderá de antemano y sin que nosotros le digamos, que el tipo de dato es de texto. De igual manera se pueden definir manualmente:

      Primitivos:

      ·undefined: tipo de dato undefined.
      ·true/false: tipo de dato booleano.
      ·2: tipo de dato numerico
      ·"string": tipo de dato string.
      ·2n: tipo de dato bigInt.
      ·Symbol() tipo de dato symbol.

      objetos:
      ·set,array tipo de dato object.

      funciones:
      ·function tipo de dato funciona.

      Expresiones regulares:
      Se utilizan para coincidir combinaciones de caracteres en cadenas. a estos en JS se los considera como objetos.

      ·Dichas expresiones se pueden crear con el valor entre barras:
      var ejemplo = /a+b/;

      ·o se puede llamar a la función RegExp "var ejemplo 2 = new RegExp('a+b');". Esta se puede cambiar según las necesidades, y pueden ser:

      test(): Buscará en una cadena y devolverá true o false.
      exec(): Prueba una coincidencia en una cadena y devolverá un arreglo de información o null.
      match(): Devolverá un arreglo con todas las coincidencias o null de caso contrario.
      replace(): Prueba una coincidencia en una cadena y reemplazará la misma por una cadena diferente.
      search(): Prueba una coincidencia en una cadena y devuelve el índice de lo encontrado, o devolverá un -1 de caso contrario.
      split(): usa la expresión regular o una cadena fija para dividirla en un arreglo de subcadenas
