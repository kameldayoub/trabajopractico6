  2. ¿Cómo se integra JavaScript a HTML?.
    • La integración de JS al HTML se puede hacer de dos formas diferentes (pero en ambos se utiliza la etiqueta "script") : -Por el body, esta forma se debe de declarar antes del cierre del body y después del contenido hecho en html: < script type="text/javascript " > var saludo = "hola "; < /script > -Por el head, se puede utiliza el atributo src para declarar la ubicación del archivo js o dentro de la etiqueta se escribe lo que se quiere hacer con el script: < script src="js/script.js">< /script> < script> console.log(); < /script>
  3. ¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?.
    • Manejo de variables:
      JS permite definir 3 tipos de variables let, const y var. La diferencia entre let y const es que el primer tipo se le puede asignar el valor y a const por otra parte no. Ejemplo:

      var numero1= 1;
      let numero2= 2;
      const numero3= 3;

      Tipos de datos:
      JS al ser un lenguaje dinámico este ya puede saber el tipo de datos según lo que contenga la variable en su interior. Si por ejemplo yo coloco la siguiente linea "var numero1= 1;" (anteriormente nombrada), sabe de antemano al contener un número, tomará el tipo de dato como numérico, lo mismo si contiene comillas "const nombre= "kamel";", entenderá de antemano y sin que nosotros le digamos, que el tipo de dato es de texto. De igual manera se pueden definir manualmente:

      Primitivos:

      ·undefined: tipo de dato undefined.
      ·true/false: tipo de dato booleano.
      ·2: tipo de dato numerico
      ·"string": tipo de dato string.
      ·2n: tipo de dato bigInt.
      ·Symbol() tipo de dato symbol.

      objetos:
      ·set,array tipo de dato object.

      funciones:
      ·function tipo de dato funciona.

      Expresiones regulares:
      Se utilizan para coincidir combinaciones de caracteres en cadenas. a estos en JS se los considera como objetos.

      ·Dichas expresiones se pueden crear con el valor entre barras:
      var ejemplo = /a+b/;

      ·o se puede llamar a la función RegExp "var ejemplo 2 = new RegExp('a+b');". Esta se puede cambiar según las necesidades, y pueden ser:

      test(): Buscará en una cadena y devolverá true o false.
      exec(): Prueba una coincidencia en una cadena y devolverá un arreglo de información o null.
      match(): Devolverá un arreglo con todas las coincidencias o null de caso contrario.
      replace(): Prueba una coincidencia en una cadena y reemplazará la misma por una cadena diferente.
      search(): Prueba una coincidencia en una cadena y devuelve el índice de lo encontrado, o devolverá un -1 de caso contrario.
      split(): usa la expresión regular o una cadena fija para dividirla en un arreglo de subcadenas
  4. ¿Cómo son todas las formas de declaración de funciones en JS?. Hay tres opciones las cuales se van acortando el código a medida de que pasan las opciones
    • A Través de las funciones declaradas function persona(){ console.log("me llamo kamel"); }
    • Se coloca dentro de una variable (y todas las variables dentro del mismo tienen que ser del mismo tipo que la de la funcion),esta funcion se la denomina como anónima ya que la misma no tiene nombre ya que al devolverse directamente a la variable, lo único que se debe de hacer es llamar a la variable. const persona = function (){ console.log("me llamo kamel"); }

  5. ¿Cuáles son los operadores y conversión entre tipos?.
    • Operadores: Hay distintos Tipos de operadores los aritmeticos, asignacion y lógicos: a) Aritméticos: Suma: var suma: 1 + 2; resta var resta: 1 - 2; multiplicación var mult: 1 * 2; división var div: 1 / 2; modulus var mod: 1 % 2; incremento var inc: inc++; decremento var dec: dec--; b)Asignación: Igual: a = b; Suma: a += b; resta a -= b; multiplicación a *= b; división a /= b; modulus a %= b; c)Logicos: igual: a==b; menor: a<=b; mayor: a>=b; desigual: a!=b; and: a&&b; or: a||b; Conversión entre tipos: Implicada: Al unir los diferentes tipos de datos JS utiliza jerarquías entre los tipos para poder definir de qué tipo de dato será el resultado. Esto dependerá del tipo y operación. Dichas conversiones se pueden convertir a string o a numero, si es una suma y hay un string el resultado sera un caracter, y si es otra operación y no hay un string, este se intentara pasar a numerio 1) cadena 2) número 3)condición 4)null 5)indefinido Explícita: O también llamado perseo utilizando la funciones Number(), String(), Boolean(). En la cual toda variable que entre , por ejemplo, en Number se va intentar convertir en número, un condicional se convertirá en 1 o 0 , y un texto si contiene algún número se convertirá en ese valor. Pero lo que pasa con Boolean es que si ponemos un carácter como "Boolean("false")" lo tomara como verdadero, en cambio, si a este no le ponemos nada y solo dejamos las comillas lo tomara como falso.Lo mismo pasará si intentamos convertir un número, la única manera que el valor sea falso será 0, el resto de números dará verdadero.

  7. ¿Cuál es el objeto dom en JS? Y ¿Cómo trabaja?.
    • Es la forma en la que JS puede acceder a nuestro documento HTML. de esta manera nosotros podremos acceder a las diferentes etiquetas para poder modificarlas. La manera de trabajar es llevando las etiquetas y sus contenidos a un documento como un grupo de nodos y objetos. Pudiendo el usuario modificar el estilo, valores y características de la página.
  8. ¿Qué función tiene en JS el Object Map?.
    • El Map es un objeto que almacena pares de clave-valores. Es similar al object, se utiliza para crear objetos, pero tiene sus diferencias. Como que las claves pueden ser de cualquier tipo, para saber el tamaño del map se utiliza la propiedad size, en cambio en un object se debe de recorrerlo para saber su tamaño. El map es iterable, además de no tener claves por defecto.
